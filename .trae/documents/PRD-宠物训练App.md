## 1. 产品概述

宠物训练助手是一款面向养狗、养猫新手主人的移动应用，帮助用户科学记录日常训练和健康习惯，通过系统化的训练课程、打卡记录和数据分析，让新手主人也能轻松培养宠物良好习惯，记录宠物成长点滴。

- 目标用户：养猫、养狗的新手宠物主人
- 核心价值：系统化训练指导 + 健康习惯养成 + 宠物成长记录
- 产品定位：温暖、专业、易用的宠物训练陪伴工具

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 宠物主人 | 手机号/微信登录 | 管理宠物档案、创建训练计划、记录打卡、查看数据报告 |
| 家庭成员 | 邀请加入 | 协作打卡、查看宠物动态、参与训练 |

### 2.2 功能模块
1. **首页计划**：今日训练概览、快速打卡入口、宠物状态卡片、本周训练进度
2. **训练课程**：课程分类浏览、分步动作说明、训练计时器、奖励记录
3. **打卡记录**：每日打卡列表、训练成效评分、异常行为备注、照片对比
4. **宠物档案**：宠物基本信息、体重曲线、疫苗驱虫记录、训练目标设置
5. **提醒中心**：训练提醒、疫苗提醒、驱虫提醒、自定义提醒
6. **每周复盘**：训练成效总结、训练建议、下周计划、周度数据统计

## 3. 核心流程

用户打开App → 选择/添加宠物 → 设置训练目标 → 浏览训练课程 → 开始训练（计时+奖励记录） → 完成打卡 → 查看训练记录 → 每周查看复盘建议

```mermaid

    A["打开App"] --> B{"有宠物档案?}
    B -->|否| C["添加宠物档案"]
    B -->|是| D["首页计划"]
    C --> D
    D --> E["选择训练课程"]
    E --> F["开始训练计时"]
    F --> G["记录奖励/异常行为"]
    G --> H["完成打卡"]
    H --> I["查看打卡记录"]
    I --> J["每周复盘建议"]
    D --> K["提醒中心"]
    D --> L["宠物档案"]
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：温暖的橙黄色系 (#FF8C42)，传递温馨、活力的感觉
- 辅助色：清新的薄荷绿 (#6BCB77)，代表健康、成长
- 背景色：柔和的米白色 (#FFFAF3)，营造温馨舒适的氛围
- 按钮风格：圆角胶囊按钮，柔和阴影，点击有微缩反馈
- 字体：圆润可爱的无衬线字体，标题粗体，正文易读
- 布局风格：卡片式布局，大圆角，柔和阴影，大量留白
- 图标风格：圆润线性图标，搭配柔和填充色
- 整体风格：温暖治愈系，可爱但不幼稚，专业但不冰冷

### 4.3 响应式
- 移动端优先设计，适配 375px - 430px 宽度
- 底部Tab导航，适合单手操作
- 触摸友好，按钮尺寸 ≥ 44px
- 支持横竖屏切换
- 关键操作触手可及区域

### 4.4 动效设计
- 页面切换：水平滑动过渡
- 按钮点击：缩放反馈 + 波纹效果
- 打卡完成：庆祝动画（撒花/弹跳）
- 计时器：数字跳动动画
- 进度条：平滑填充动画
- 卡片加载：渐入 + 轻微上浮
